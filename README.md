This project will implement a shape matching system that identifies and compares objects based on their geometric structure, while also analyzing robustness under transformations and noise. 
Given input images, the system will segment objects, extract shape descriptors using similitude moments, and compute similarity between shapes. 
In addition, the project will test how well the system performs under real-world variations such as rotation, scaling, translation, and noise. 
The goal is to demonstrate both shape recognition and the robustness of these descriptors under varying image conditions.
